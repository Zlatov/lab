Настройка

Изменение языка ввода на экране приветствия Windows 7

  1. Панель управления → Часы, язык и регион → Изменить язык интерфейса → Язык и клавиатуры → Изменить клавиатуру → Язык ввода по умолчанию (выбираем желаемый язык по умолчанию)
  2. Панель управления → Часы, язык и регион → Язык и региональные стандарты → Дополнительно → Копировать параметры (ставим галки чтобы сопировать язык по умолчанию остальным)

RDP, Подключение к удаленному рабочему столу

Пуск → Все программы → Стандартные → Подключение к удаленному рабочему столу, Компьютер: <mydomain>.ddns.net:<myport>
