Псевдоклассы

По номеру элемента

nth-child

Добавляет стиль к элементу номер которого входит в числовую последовательность описанную параметром псевдокласса формулой вида an+b.

Пример:

.my:nth-child(2n-3) {
	color: red;
}

Все нечетные элементы (номер элемента назначется сугубо в одном уровне дум - элементы имеющие одного родителя) класса .my будут окрашены в красный цвет.

nth-last-child

То же что и предыдущее только отсчет с конца.

nth-of-type

То же что и nth-child, но учитывается тип тега

Пример:

span:nth-child(2n-3) {
	color: red;
}

Посчитываться будут только span

nth-last-of-type

То же что и предыдущее, но подсчет с конца.

Медиа запросы

min-width: 768px === Правило применяется к браузеру, ширина которого больше значения, указанного в запросе.

max-width: 1024px === Правило применяется к браузеру, ширина которого меньше значения, указанного в запросе.

/*@media (min-width: 768px) and (max-width: 991px) {*/
@media (min-width: 768px) {
    .navbar-nav li:first-child a {
        padding-left: 0;
    }
}

Флекс боксы

Стили для контейнера

.flex {
	display: flex;

	/* Ось */
	/* расположить боксы горизонтально */
	flex-direction: row;
	/* вертикально */
	flex-direction: column;

	/* Выравнивание вдоль оси (подобно text-align если боксы горизонтально) */
	justify-content: flex-start;
	justify-content: flex-end;
	justify-content: center;
	justify-content: space-between;
	justify-content: space-around;

	/* Выравнивание поперек оси */
	align-items: stretch;
	align-items: flex-start;
	align-items: flex-end;
	align-items: center;
	align-items: baseline;

	/* Многострочность */
	flex-wrap: nowrap;
	flex-wrap: wrap;

	/* Выравнивани многострочности вдоль оси многострочности (поперек главной оси) */
	align-content: stretch;
	align-content: flex-start;
	align-content: flex-end;
	align-content: center;
	align-content: space-between;
	align-content: space-around;
}
