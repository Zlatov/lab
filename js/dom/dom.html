Как копировать колонку таблицы из браузера

  1. Скопировать «инструмент» в буфер обмена и вставить в консоль браузера: 
    function get_table_column(table_selector, column_index) {
  return [].slice
    .call(
      window.document.querySelectorAll(table_selector)[0]
        .querySelector("tbody")
        .getElementsByTagName('tr')
    )
    .map(tr => tr.getElementsByTagName('td'))
    .filter(td => td.length > 0)
    .map(td => td[column_index].innerHTML)  
}
get_table_column("", 0)
  2. Найти тег таблицы, копировать селектор тега (Copy selector), вставить селектор первым параметром функции (последняя строка кода);
  3. Вторым параметром указать правильный index столбца. Запустить код.
