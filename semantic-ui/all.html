Установка

bower info semantic-ui - узнать последнюю версию.

bower i semantic-ui#2.2.7 -S

bower list --path - получить ссылки:

  semantic: [
    'theme/bower/semantic/src/semantic.less',
    'theme/bower/semantic/dist/semantic.js'
  ]

Для css на самом деле есть файл по пути:

    'theme/bower/semantic/dist/semantic.css'
    # или
    'theme/bower/semantic/dist/semantic.min.css'

Примерно так:

		<link rel="stylesheet" type="text/css" href="/theme/bower/semantic/dist/semantic.min.css">
		<script type="text/javascript" src="/theme/bower/semantic/dist/semantic.min.js"></script>

Либо можно сконвертить свой scc из less...
