Bower — пакетный менеджер для web.

Установка

sudo npm install -g bower

проверка:

bower -v
/usr/bin/env: node: No such file or directory
sudo ln -s "$(which nodejs)" /usr/bin/node
bower -v
1.8.0

Команды

bower s jquery - поиск пакета.

bower info jquery - информация о пакете.

bower i jquery#3.1 - установка пакета.

bower list - список установленных пакетов.

bower list --path - список путей пакетов.

bower uninstall jquery - удаление пакета.

Управление пакетами

bower init - создание bower.json

bower i jquery#3.1 -S (--save) - добавлние пакета в production

bower i jquery#3.1 -D (--save-dev) - добавлние пакета в dev

bower uninstall jquery -S (-D) - удаление пакета с указание типа.

bower update jquery - обновление пакета.

Версии пакетов

"jquery": "latest" - последняя версия.

#2.1.4 - 2.1.4.

#~2.1.4 - 2.1.4 <= версия < 2.2.0.

#^2.1.4 - 2.1.4 <= версия < 3.0.0.

Место установки

Указывается в файле .bowerrc

touch .bowerrc

subl .bowerrc

{
  "directory" : "theme/bower"
}

В файл subl .gitignore добавить theme/bower.

Объединить bower и composer

Каждый запуск composer install или composer update будет выполняться запуск bower install, который обновит все библиотеки.

Добавляем в composer.json

{
  // ...
  "scripts": {
      "post-install-cmd": [
      "bower install"
    ],
      "post-update-cmd": [
      "bower install"
    ]
  }
}
